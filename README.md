### AIM: To write a C program to print numbers from 1 to n(a given number
---
### ALGORITHM :
1.Get a input number n from the user.
2.using for loop, print the numbers from 1 to n.
---
### PROGRAM:
```
#include<stdio.h>
int main()
{
    int num;
    scanf("%d",&num);
    for(int i=1;i<=num-1;i++){
        printf("%d, ",i);
    }printf("%d",num);
    return 0;
}
```
